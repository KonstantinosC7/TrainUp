@startuml
' ============ Κλάσεις Domain =============
class User
class Student
class Professor
class Company
class Committee
class TraineeshipPosition
class Evaluation

' ============ Κλάσεις Service =============
class UserService
class CompanyService
class StudentService
class ProfessorService
class CommitteeService

' ============ Κλάσεις Strategies =============
class InterestsBasedSearchStrategy
class LocationBasedSearchStrategy
class InterestsBasedSupervisorStrategy
class LoadBasedSupervisorStrategy

' ============ Κληρονομικότητα =============
User <|-- Student
User <|-- Professor
User <|-- Company
User <|-- Committee

' ============ Συσχετίσεις Domain =============
' (1) Μια εταιρεία μπορεί να προσφέρει 0..* θέσεις πρακτικής
Company "1" -- "0..*" TraineeshipPosition

' (2) Ένας καθηγητής μπορεί να επιβλέπει 0..* θέσεις πρακτικής
Professor "1" -- "0..*" TraineeshipPosition

' (3) Ένας φοιτητής μπορεί να έχει 0..1 θέση πρακτικής
Student "1" -- "0..1" TraineeshipPosition

' (4) Μια θέση πρακτικής μπορεί να έχει 0..* αξιολογήσεις
TraineeshipPosition "1" -- "0..*" Evaluation

' ============ Χρήση των Services =============
UserService ..> User
CompanyService ..> Company
CompanyService ..> TraineeshipPosition
StudentService ..> Student
StudentService ..> TraineeshipPosition
ProfessorService ..> Professor
ProfessorService ..> TraineeshipPosition
CommitteeService ..> Committee
CommitteeService ..> TraineeshipPosition

' ============ Χρήση των Strategies =============
CommitteeService ..> InterestsBasedSearchStrategy
CommitteeService ..> LocationBasedSearchStrategy
CommitteeService ..> InterestsBasedSupervisorStrategy
CommitteeService ..> LoadBasedSupervisorStrategy
@enduml
